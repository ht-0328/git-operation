# 操作コマンド一覧

## Docker の操作コマンド

### コンテナの一覧を表示する

```sh
docker ps -a
```

### Docker の未使用データをすべて削除する

```sh
sudo docker system prune -a
```

## Git の設定方法

### ユーザ名の設定

```sh
git config --global user.name "自身のユーザ名"
```

### メールアドレスの設定

```sh
git config --global user.email "自身のメールアドレス"
```

## VSCode コマンド

"Ctrl + Shift + P"でコマンドパレット開き、各コマンドを実行する
